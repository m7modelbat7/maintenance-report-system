# AGENTS.md

## Frontend goal
Build a simple, clean, professional React interface for the maintenance report system.

## Frontend rules
- Use functional React components
- Keep components small and clear
- Prefer simple state management
- Keep forms readable
- Handle loading, empty, and error states
- Do not introduce unnecessary complexity

## UI priorities
- Clean and professional
- Easy for technicians and supervisors
- Tables and forms should be clear
- Start simple, improve later

## Before finishing
- Explain the component flow
- Explain how to test manually
- Check that no obvious UI state is missing
