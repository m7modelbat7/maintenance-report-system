# Review Check Skill

## Goal
Review work before moving to the next step.

## Check for
- bugs
- missing validation
- poor naming
- weak error handling
- missing loading/empty/error UI states
- unnecessary complexity

## Output format
- findings
- severity
- recommended fixes
