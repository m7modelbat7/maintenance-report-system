# Backend CRUD Skill

## Goal
Implement backend work in a simple, readable, beginner-friendly way.

## Instructions
1. Start from the data model.
2. Keep routes clear and predictable.
3. Validate inputs.
4. Return consistent JSON responses.
5. Handle errors clearly.
6. Add or suggest the smallest useful test.

## Deliver
- files changed
- what each file does
- how to run
- how to test
