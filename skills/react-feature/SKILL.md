# React Feature Skill

## Goal
Build simple, clean UI features that are easy for a beginner to understand.

## Instructions
1. Understand the user flow first.
2. Keep component structure simple.
3. Handle loading, empty, and error states.
4. Keep forms controlled and readable.
5. Avoid unnecessary libraries.
6. Explain manual testing steps.

## Deliver
- components created or updated
- UI state explanation
- how to test manually
